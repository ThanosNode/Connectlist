🧩 API & Data Flow
Ensure all API endpoints follow consistent REST principles and return structured success/error responses.

Validate all API request payloads and query params using Zod on the frontend and backend.

🗃️ Database & Drizzle ORM

Add connection pooling if not present.

Use Drizzle’s type-safe queries to ensure no raw or unsafe SQL operations exist.

Implement retry logic for transient database failures.

Sanitize all inputs at both application and database layers.

🔄 WebSocket Handling (if active)
Harden partial WebSocket implementation.

Add reconnection logic, ping/pong handling, and server-side failover where applicable.

🔗 Frontend Routing & Links

Add 404 fallbacks for unmatched routes.

🧪 Forms & Validation
Ensure all forms use proper schema validation via Zod and React Hook Form.

Verify Express Session and Passport.js are configured securely (cookies, CSRF, session expiration).
